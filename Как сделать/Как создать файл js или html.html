Как создать файл js или html

В целом, логика создания js, html или любого другого файла с определенным расширением - одна.

Самый простой способ - создать файл через "проводник".

Но для этого нужно сначала включить опцию чтобы мы могли видеть расширение файлов (по умолчанию оно скрыто, особенно на компьютерах колледжа).

Ну а дальше уже все просто, например на нужно создать js файл example.js
